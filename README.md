### Running
To run you need to have rust installed, [here is rusts docs on how to do that](https://rust-lang.org/tools/install/)

Then you just `cd` into the day you want to run and use `cargo run`, like day 1 for example:

```bash
cd <path to this repo>/day-1 && cargo run
```
